# BO4GreenSolvent

This code is about the two-step BO framework for development of TEP-based solvent for air-processed perovskite solar cells. 


Some key packages (and their versions) involved to run this code are as follows:

GPy 1.10.0

Emukit 0.4.9

shap 0.41.0

paramz 0.9.5

numpy 1.23.5 

emcee	3.1.2

scikit-learn 0.23.2
